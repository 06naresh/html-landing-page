# HTML Landing Pages

This repository contains a collection of **HTML-only landing pages**.  
Each file represents a different landing page design, built without external CSS or JavaScript frameworks.

## 📂 Contents
- Individual HTML files for each landing page
- Documentation explaining the structure and purpose of each page

## 🎯 Purpose
The goal of this repository is to practice and demonstrate web page structuring using pure HTML.  
It serves as a portfolio of landing page designs and a foundation for future projects that may include CSS and JavaScript enhancements.
